# Registration Form

A full-stack registration form built with **React**, **Express**, and **MySQL**.

## Features

- Registration form with validation
- Data stored in MySQL database
- View all submissions in the app
- MySQL Workbench compatible

## Quick Start

### 1. Setup MySQL

Run `backend/database/schema.sql` in MySQL Workbench.

### 2. Backend

```bash
cd backend
npm install
copy .env.example .env
# Edit .env and set DB_PASSWORD
node server.js
```

### 3. Frontend

```bash
cd frontend
npm install
npm run dev
```

Open **http://localhost:3000**
